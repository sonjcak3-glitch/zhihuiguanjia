# 智绘管家 - AI创作平台

一个高端、现代化的AI创作平台，提供专业级的图片与视频创作功能。

## 功能特性

### 图片创作
- **文生图**：通过文字描述生成高质量图片
- **图生图**：基于参考图片进行创意转换

### 视频创作
- **产品视频**：专业的产品展示视频生成
- **超级播客**：支持场景图+人物图的播客视频制作

## 设计特点

- 🎨 **高端设计**：参考苹果官网的设计语言，简洁大气
- ✨ **流畅交互**：丝滑的动画效果和过渡
- 📱 **响应式布局**：完美适配桌面和移动设备
- 🎯 **用户体验**：直观的操作流程，专业的视觉呈现

## 技术栈

- HTML5
- CSS3（现代CSS特性）
- 原生JavaScript（无框架依赖）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 主要功能说明

### 导航系统
- 顶部固定导航栏，支持快速切换不同板块
- 智能高亮当前页面
- 平滑滚动效果

### 创作工具
- **文生图**：输入创意描述，调整参数，一键生成
- **图生图**：上传参考图，设置相似度，转换风格
- **产品视频**：上传产品图片，选择风格和时长
- **超级播客**：上传场景和人物图，选择语音，生成播客视频

### 交互特性
- 拖拽上传文件
- 实时参数调整
- 键盘快捷键支持（ESC返回首页，Ctrl+Enter生成）
- 自动保存功能
- 通知提示系统

## 自定义配置

### 修改主题色
在 `styles.css` 中修改CSS变量：

```css
:root {
    --primary-color: #0071e3;  /* 主色调 */
    --secondary-color: #1d1d1f;  /* 次要色 */
    /* ... 其他颜色变量 */
}
```

### 调整布局
所有响应式断点都在 `styles.css` 的媒体查询中定义，可根据需要调整。
